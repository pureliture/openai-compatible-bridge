Ollama Per-Request Reasoning Requirements

Source of truth: requirements.md

Bridge는 기존 OLLAMA_THINK 기본값을 유지하면서, OpenAI-compatible reasoning_effort와 reasoning.effort 요청 필드로 Ollama think 값을 호출별 override해야 한다.

핵심 결정

  • Provider endpoint는 현재 native /api/chat 경로를 유지한다.
  • high, medium, low는 동일 문자열로 전달한다.
  • none은 명시 요청일 때만 think=false로 전달한다.
  • invalid 값은 upstream 호출 전 HTTP 400으로 거부한다.

승인

이번 턴에서 사용자가 자문자답 및 승인 진행을 위임했다. 승인 기준은 requirements.md다.
